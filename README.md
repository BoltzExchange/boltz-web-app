# Boltz Web App

This is the source of the official Boltz Web App served at [boltz.exchange](https://boltz.exchange/). It enables **non-custodial** swaps between different Bitcoin layers.

## Documentation

We encourage our technical users to check the code and run the web app locally from source following [these instructions](https://docs.boltz.exchange/v/web-app/) in our docs.
